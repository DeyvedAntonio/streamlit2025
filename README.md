# Streamlit

## Criando uma aplicação de controle financeiro
Com base no curso [Streamlit Relâmpago](https://www.youtube.com/playlist?list=PLvlkVRRKOYFRYA40hJ_V8e_iC5Lu6YPyn) do [TeoMeWhy](https://github.com/TeoMeWhy).

## Como iniciar o projeto
- Instalação do [Anaconda](https://anaconda.org/);
- Instalação da IDE de sua preferência;
- Criar o Ambiente Virtual e informa qual versão do Python será usada:
  `conda create --name Streamlit-2025 python=3.13`
- Ativar o Ambiente Virtual:
  `conda activate Streamlit-2025`
- Instalar as dependências do projeto:
  `pip install streamlit`
- Iniciar o servidor do Streamlit:
  `streamlit run main.py`
